📊 HR Analytics Dashboard — Project Overview
🎯 Purpose
The dashboard is designed to provide actionable insights into workforce data, enabling HR teams and leadership to make data-driven decisions that improve employee engagement, retention, and organizational performance.
🔍 Key Focus Areas
Employee id, department, experiance, rating, country.
FEATURES
interative excel dash board, KPI cards, slicers for dynamic filtering, Pivot charts and Pivot tables.
Clean and userfriendly dash board layout.
📈 Expected Outcomes
Better visibility into workforce dynamics.
Identification of problem areas (e.g., high turnover in specific teams).
Improved recruitment strategies through data-backed insights.
Enhanced employee satisfaction and retention.
Support for strategic HR planning and policy-making.
